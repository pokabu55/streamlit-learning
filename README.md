# Usage

* Streamlit の開始方法
```bash
git clone [this repository]
cd streamlit-learning
cd streamlit-learning/modules
make
cd ..
streamlit run xxx.py
```

* 必要なC++アプリのビルド方法
```bash
cd ./modules/{ビルドしたいモジュール}
make
```
